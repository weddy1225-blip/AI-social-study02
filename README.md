## 📖 遊戲背景
在 2077 年的世界，人類的歷史檔案遭到損壞。AI 助理雖然能掃描出古物的外觀，卻因為缺乏生活經驗而產生各種爆笑的誤解（例如：把撥盤式電話當成舉重啞鈴）。身為專業考古隊員的學生，必須從四個選項中鑑定出正確的名稱，守護家鄉的記憶。

## ✨ 遊戲特色
- **沉浸式劇本**：獨立的開場與結局分頁，增強學生的角色扮演感。
- **動態題庫**：內建 100 組精選古物題庫，每次遊戲隨機抽選 6 題，百玩不膩。
- **護眼視覺設計**：
  - 採用深色背景（Dark Mode）搭配柔和綠色與米白文字，減輕教室投影幕對眼睛的負擔。
  - 加大加粗字體與按鈕，適合平板與觸控螢幕操作。
- **階層式結局**：根據鑑定成功率觸發不同的結局（歷史修正官 / 時空詐欺師 / 文明斷層者）。

## 🛠️ 技術架構
- **開發語言**：Google Apps Script (JavaScript / HTML / CSS)
- **部署平台**：Google Web App
- **硬體需求**：平板、桌機、筆電或具備網頁瀏覽器的電子白板。

## 📥 如何部署到自己的 Google 帳號
1. 前往 [Google Apps Script](https://script.google.com/)。
2. 點擊「新專案」。
3. 分別建立 `Code.gs` 與 `index.html` 檔案，並貼上本專案的原始碼。
4. 點擊右上方「部署」 > 「新部署」。
5. 類型選擇「網頁應用程式」，並設定「所有人」皆可存取。
6. 完成部署後，將得到的網址分享給學生即可開始。
